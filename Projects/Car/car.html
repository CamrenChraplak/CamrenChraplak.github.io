Arduino Car Interface

(5/2024 to Present)

 This is a personal project I created to add additional features to my current vehicle. Currently, it doesn't have any headphone input for playing music and doesn't have a functioning speedometer. This project aims to fix this by adding connections for connecting a headphone jack, adding play/pause/forward/reverse button controls, and adding a digital spedometer.


My overall goals for this project is to improve my car by

  • having built in support for headphone jack and audio controls.
  • having built in support for a digital speedometer.
  • encasing and connecting modules with the car.


The status of my goals for the project include

  • developing and fabricating PCBs.
  • creating fully functional code for the project.
  • testing speedometer PCB functionality.


The goals that are still a work in progress include

  • testing the audio PCB.
  • creating 3D prints for housing the modules.
  • connecting it to my vehicle.
