Microcontroller Oscilloscope

(6/2025 to Present)

 Oscilloscopes in the modern age can cost hundreds of dollars for a basic model while it is often a necessity for any complex electronics project. To solve this, many tinkerers have developed their own oscilloscopes, with many posted online. However, most solutions have low data collection rates, lack common oscilloscope features, and require specialized hardware. The goal of the project is to provide open source access to an oscilloscope, with the only necessary hardware being a micro-controller and a phone/computer, along with providing the functionality of most common oscilloscopes.


My overall goals for this project is to create an accessible oscilloscope by

  • enabling makers to use microcontrollers they have on hand at a cheap price.
  • providing WiFi, Bluetooth, and Serial connections for various communication methods.
  • including addons for extra features (ie. external ADC's, AC voltage, logic analyzer, function generator, etc.).


The status of my goals for the project include

  • developing code that is board independent using PlatformIO.
  • configurable code that allows custom user functionality for essential features.
  • implimenting unit tests for common board functionality.


The goals that are still a work in progress include

  • finishing base oscilloscope functionality for reading and sending voltages.
  • implementing board optimized configurations for faster data collection rates.
  • developing communication apps between the microcontrollers and monitoring devices (laptops, phones, etc.).
